Módulo 1: Los Primeros Pasos

¿La programación es para mí? Descubre tu potencial, entiende la lógica y prepara tu mente.

Módulo 2: Herramientas del Oficio

Configura tu entorno. Conoce los editores de código, la terminal y el control de versiones.

Módulo 3: Fundamentos de la Web

Aprende a construir la estructura y el estilo de las páginas web with HTML5 y CSS3.

Módulo 4: El Poder de JavaScript

Dale vida a tus sitios web. Aprende variables, funciones, eventos y manipulación del DOM.

Módulo 5: Lenguajes para Empezar

Módulo 6: Tu Primer Proyecto

Tu Caja de Herramientas Digital

Un buen programador conoce sus herramientas. Estos son los editores de código más populares.

Visual Studio Code

El estándar de la industria. Gratuito, de código abierto y con un ecosistema masivo de extensiones para cualquier necesidad.

Sublime Text

Conocido por su velocidad y eficiencia. Es ligero, rápido y altamente personalizable. Ideal para quienes aman el minimalismo.

Replit

Programa desde tu navegador. No requiere instalación y es perfecto para aprender, colaborar y prototipar rápidamente.

Ejercicio 2: Estiliza tu Botón

Usando CSS, crea un botón que tenga un fondo de color púrpura, texto blanco, esquinas redondeadas y que cambie de color cuando pases el mouse sobre él.
